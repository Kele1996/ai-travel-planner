# AI Travel Planner ✈️

An AI-powered frontend web application that helps users plan personalized trips based on their destination, travel dates, budget, and interests.

## Project Goal

This project is my FlyRank AI Frontend Development Capstone.

The goal is to demonstrate modern frontend engineering practices by building a responsive, accessible, and production-quality React application.

## Features (Planned)

- Beautiful landing page
- Trip planning form
- AI-generated travel itinerary
- Weather forecast
- Budget breakdown
- Packing checklist
- Interactive map
- Responsive design
- Dark mode
- Loading and error states

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS Modules
- React Router
- Fetch API
- OpenAI API (or another LLM API)
- Weather API
